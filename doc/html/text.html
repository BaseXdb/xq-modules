Higher-Order XQuery Libraries

Module List

base64 A module for converting between byte sequences and xs:base64Binary values.
heap A simple Skew Heap implementation.
int-set Abstract interface for a set of integers.
int_set/jpcs Implementation of a set of integers based on John Snelson's Red-Black Tree.
int_set/lw Implementation of a set of integers based on a Red-Black Map.
int_set/rbtree_jpcs
list A linked list.
map2 Additional functions for XQuery 3.
ordered-map Abstract interface for an ordered map.
ordered_map/avltree-simple Simple implementation of an ordered map based on an AVL Tree.
ordered_map/avltree Implementation of an ordered map based on an AVL Tree.
ordered_map/rbtree Implementation of an ordered map based on a Red-Black Tree.
pair A library for typed pairs of XQuery sequences.
queue A simple queue implementation.
rng A simple random number generator implemented as a Linear Congruential Generator.
utf8 Module for encoding and decoding xs:string s as xs:byte sequences using the UTF-8 encoding scheme.
