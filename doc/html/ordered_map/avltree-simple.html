Library Module: ordered_map/avltree-simple.xqm

URI: http://www.woerteler.de/xquery/modules/ordered-map/avltree-simple
Description: Simple implementation of an ordered map based on an AVL Tree.
Author: Leo Woerteler <leo@woerteler.de>
Version: 0.1

Functions

check($lt, $tree, $min, $max, $msg)

Arguments:
$lt item()* less-than predicate
$tree item()* current node
$min item()* key that is smaller than all keys in $tree
$max item()* key that is greater than all keys in $tree
$msg item()* error message to show when an invariant is violated
Returns:
item()* empty sequence
Annotations:
%public
Description: Checks the given tree node for invariant violations.
Error: rbtree:AVLT0001 if a key is smaller than $min
Error: rbtree:AVLT0002 if a key is greater than $max
Error: rbtree:AVLT0003 if the AVL Tree is unbalanced
